🧩 Projeto: Sistema de Cadastro de Clientes e Serviços de TI
📘 Descrição

Este projeto será uma aplicação Full Stack desenvolvida com Java 8 (Spring Boot) no backend e Angular no frontend.
O sistema tem como objetivo cadastrar clientes e registrar os tipos de serviços de TI realizados para cada um deles.

A aplicação permitirá o gerenciamento de clientes, o registro dos serviços prestados e a visualização de históricos de atendimento, de forma simples e eficiente.
